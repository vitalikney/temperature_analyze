# Проект 1 — анализ температур

Небольшое Streamlit‑приложение для анализа исторических температур, поиска аномалий и сравнения с текущей погодой через OpenWeatherMap.

## Как запустить
1. Установить зависимости:
   ```bash
   python -m pip install -r requirements.txt
   ```
2. Запустить:
   ```bash
   streamlit run app.py
   ```
3. Либо загрузить `temperature_data.csv` в интерфейсе, либо положить его рядом с `app.py`.

## Выводы по параллельности и async
- Параллельный анализ (потоки) даёт небольшой выигрыш только при большом объёме данных. На небольших наборах данных разница почти незаметна.
- Для одного города проще синхронный запрос к API: меньше кода и всё понятно.
- Асинхронный подход полезен, когда нужно делать много запросов одновременно (например, сразу по десяткам городов).
